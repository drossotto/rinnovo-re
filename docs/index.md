# Rinnovo Binary (RNB)

Rinnovo Binary (RNB) is a single-file, segment-based container format
and runtime interface for biological datasets.

This documentation site covers:

- the RNB file format and segment specifications;
- the manifest and query kernel model;
- usage examples and tooling (to be added).

For the current on-disk format details, see
the RNB Format v0.1 specification.
