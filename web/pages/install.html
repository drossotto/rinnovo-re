Install a local Rinnovo engine

The Rinnovo engine runs next to your data and exposes a small HTTP API that the console connects to. This page describes a simple development setup using the open-source repository.

1. Clone the repository

In a terminal with Rust and Cargo installed:

git clone https://github.com/drossotto/rinnovo-re.git
cd rinnovo-re

2. Run the engine HTTP service

Build and run the Axum-based HTTP wrapper around the engine runtime: 

cargo run -p rnb_engine_http

By default this listens on http://127.0.0.1:8787.

3. (Optional) Register with a registrar

Once a registrar service is running, the engine will be able to register itself using an environment variable: 

export RINNOVO_REGISTRAR_URL=<https://rinnovo-registrar.example.com>

The console can then discover your engine and workspace via the registrar instead of hard-coding URLs.
